# Saturday Systems Website

The official website for Saturday Systems - helping high-achievers reclaim their time.

## About

Saturday Systems designs automations that cut out busywork, allowing you to focus on what matters most.

**Mission**: Work less, live more.

## Website

- **Production**: [buildsaturday.com](https://buildsaturday.com)
- **GitHub Pages**: Available via GitHub Pages

## Technology

- Pure HTML, CSS, and vanilla JavaScript
- No build process required
- Responsive design
- Hash-based navigation for single-page experience

## Local Development

Simply open `index.html` in a browser, or run a local server:

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Then visit `http://localhost:8000`

## Deployment

This site is automatically deployed via GitHub Pages. Any push to the `main` branch will update the live site.
